Get running in 5 minutes.

### Prerequisites
- Docker & Docker Compose
- Python 3.13+
- uv or pixi (recommended) or pip

```bash
# 1. Clone repository

cd praviel

# 2. Start services (PostgreSQL, Redis, Qdrant)

# 4. Launch
uvicorn app.main:app --reload
# Visit http://localhost:8000
```
